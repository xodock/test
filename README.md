# Test project for Solar Digital
Dev deploy instruction:

- git clone https://github.com/xodock/test.git 
- do "composer install"
- do "npm install"
- do "npm run dev"
- setup your server environment(web-server, MySQL, php)
- do "npm run migrate"

Backend Tests run:
-
- nix: npm run backtestnix
- win: npm run backtestwin
  
UI Tests run:
-
- npm run uitest
